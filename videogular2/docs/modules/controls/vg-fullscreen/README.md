# VgFullscreen @Component

Button to toggle between fullscreen and normal mode.

## HTML Definition

```html
<vg-player>
    <vg-controls>
        <vg-fullscreen></vg-fullscreen>
    </vg-controls>

    <video #myMedia
           [vgMedia]="myMedia"
           id="my-video"
           src="http://static.videogular.com/assets/videos/videogular.mp4"
           type="video/mp4">
    </video>
</vg-player>
```
